# Recomp Studio — guía completa para crear una página nueva en GitHub

## Qué vas a crear

Una aplicación nueva y separada de tu repositorio anterior.

Nombre sugerido del repositorio:

`recomp-studio`

La dirección final tendrá esta estructura:

`https://TU-USUARIO.github.io/recomp-studio/`

## 1. Descargar y descomprimir

1. Descargá `recomp-studio-github.zip`.
2. Abrilo en tu computadora.
3. Vas a ver una carpeta llamada `recomp-studio`.
4. Entrá en esa carpeta.

Los archivos principales deben verse así:

- `.nojekyll`
- `index.html`
- `styles.css`
- `app.js`
- `routine-data.js`
- `manifest.json`
- `service-worker.js`
- `README.md`
- `icons/`

No subas el ZIP cerrado.

## 2. Crear un repositorio nuevo

1. Entrá a GitHub.
2. En la esquina superior derecha, presioná `+`.
3. Elegí `New repository`.
4. En `Repository name`, escribí `recomp-studio`.
5. Elegí `Public`.
6. No marques `Add a README`, `.gitignore` ni licencia.
7. Presioná `Create repository`.

## 3. Subir la app

En la pantalla vacía del nuevo repositorio:

1. Presioná el enlace `uploading an existing file`.
2. Abrí la carpeta descomprimida `recomp-studio`.
3. Seleccioná todo lo que está adentro.
4. Arrastrá los archivos a GitHub.
5. Confirmá que `index.html` aparezca directamente en el nivel principal.
6. En el mensaje de commit escribí `Primera versión de Recomp Studio`.
7. Presioná `Commit changes`.

## 4. Activar GitHub Pages

1. Dentro del repositorio, abrí `Settings`.
2. En el menú lateral, abrí `Pages`.
3. En `Source`, elegí `Deploy from a branch`.
4. En `Branch`, elegí `main`.
5. En la carpeta, elegí `/(root)`.
6. Presioná `Save`.

Después de unos minutos, GitHub mostrará la URL de la página.

## 5. Instalar en iPhone

1. Abrí la URL publicada en Safari.
2. Presioná el botón Compartir.
3. Elegí `Agregar a pantalla de inicio`.
4. Confirmá con `Agregar`.

## Privacidad

La rutina y el código de la app serán visibles porque el repositorio es público.
Los pesos, repeticiones, notas e historial se guardan localmente en el navegador del dispositivo y no se suben al repositorio.
